## Contributing

Contributions are welcome!

### Prepare

- clone the repo
- run `npm install`

### Run the explorer

- run `npm run dev`

### Add new projects

- add new projects to `src/lib/kube/projects.js`
- run `npm run download` (i)
- run `npm run build`

(i) A GitHub Access Token is required. Please set the `GITHUB_TOKEN` environment variable.
